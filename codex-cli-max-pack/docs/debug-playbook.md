# Debug Playbook

## 1. Symptom
- 何が起きているか
- 期待値は何か
- どこで観測したか

## 2. Reproduction
- 再現手順
- 必要な入力
- 環境条件
- 再現率

## 3. Evidence
- エラーメッセージ
- スタックトレース
- ログ
- テスト失敗内容
- 関連 diff / recent change

## 4. Hypotheses
- 仮説A
- 仮説B
- 仮説C

## 5. Fastest Checks
- 最も安い確認から順に試す
- 仮説を潰すための追加ログ / テスト / 読解

## 6. Root Cause
- 直接原因
- なぜその状態になったか
- なぜ既存テストが防げなかったか

## 7. Fix
- 最小修正
- 回帰テスト
- 副作用確認
