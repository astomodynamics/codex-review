# Execution Plan Template

## 1. Goal
- 何を実現するか
- 何はやらないか

## 2. Constraints
- 互換性
- 速度 / 安全性 / 運用上の制約
- 触ってよい範囲 / 触らない範囲

## 3. Impacted Areas
- Entry points
- Business logic
- Data / schema
- Tests
- Docs / config

## 4. Unknowns / Risks
- 不明点
- 仮説
- 破壊ポイント
- ロールバック観点

## 5. Plan
1. 調査
2. 最小変更
3. 追加修正
4. 検証
5. 仕上げ

## 6. Validation
- format / lint
- typecheck / build
- unit / integration / e2e
- manual verification

## 7. Done When
- 完了条件を箇条書き
